# fuel codes
```
B7 = Diesel containing up to 7% biodiesel (FAME)
Most standard diesel on UK forecourts is B7.
```

```
B7S = B7 diesel intended for summer-grade use
The S usually means summer specification.
```

```
B7P = B7 diesel with a different seasonal or premium specification
The P is commonly used by some suppliers for premium or protected winter spec, depending on the distributor. It is not a universal standard marking like B7 itself.
```

```
B10 = Diesel containing up to 10% biodiesel
Used mainly in some fleets/commercial vehicles.
Not all diesel engines are approved for B10.
```

```
HVO = Hydrotreated Vegetable Oil diesel
A synthetic/paraffinic renewable diesel alternative.
Often branded as “renewable diesel”.
Can significantly reduce lifecycle CO₂ emissions compared with conventional diesel.
```
